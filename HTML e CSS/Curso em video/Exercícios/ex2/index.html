Para fazer aparecer < ou > na página do HTML, é preciso digitar: & + (lt ou gt) + ;

Símbolos especiais: copyright ©, registered ®, trademark ™, euro €, yen ¥, libra £, delta Δ, seta para cima ↑.

Emojis: 🖖, 😀, 👻, 🐉. Para adicionar um emoji é preciso digitar: &#x + ID do emoji.
