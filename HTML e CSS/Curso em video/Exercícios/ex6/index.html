Lista de definições

HTML
Não é uma linguagem de programação, é responsável pelo "esqueleto" de uma página web,
ou seja, sua estrutura.
CSS
Responsável pela aparência da página web, também não é uma página web.
JavaScript
Linguagem de programação responsável pela interatividade e funções da página web.
